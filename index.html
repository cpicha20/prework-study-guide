HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Metadata is information that helps the browers render the page correctly.
  • Metadata isnt shown to the user
  • HTML is used to structure the webpage and provide a framework

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A . (period) designates a class selector, which allows us to assign class atributes to specific elements.
  • Linking external style sheets should be done in the metadata, or head, section of an html file.
  • Selector defines the elements that declarations will apply to.
  • Declarations are made up of 2 components, a property we want to apply and the value of that property. For example the color property has the value of blue.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: modifies the current branch to the staging area, middle ground between local file and what you had last commmited. -A: commits all files
  • git commit: pushes changes to the branch, you can revert changes
  • git pull: creates a pull request to merge branches

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a powerful programming language that can add interactivity to a website
  • A variable is a named container that allows us to store data in our code. We assign variables using the "var" keyword.
  • js files are linked in the body, below the footer
  • When encountering an "if" statement in a JavaScript file, it will check to see "if" the condition that we provide is truthy.
  • An array is a single variable that is used to hold a group of data using the [] symbols.
  • An index is a location of a object within an array that is identified with a unique number.
  • A "for" loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
